# pentacle css 

A super simple css framework written in sass.

Designed to provide a very clean layout that just works.

Currently in alpha. 

February 2021.



## Use

$ yarn add sass

$ yarn add lottecss



### Add the following line of code to App.scss

@import '~lottecss/assets/styles/lotte.scss';



### In App.js

import './App.scss';
